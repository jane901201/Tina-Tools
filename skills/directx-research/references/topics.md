# DirectX Topics

## Environment Checklist
- API layer (D3D11 or D3D12)
- Feature level and optional capabilities
- GPU model, driver version, OS build

## Pipeline Validation
- Resource state transitions and barriers
- Descriptor heap layout and lifetime
- Command list recording and submission order

## Debug and Profiling
- Enable debug layer and capture validation messages
- Use PIX captures to isolate stalls and bubbles
- Check shader permutations, PSO creation, and cache behavior
